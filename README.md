# Shop

My first fullstack project with Express.js (Backend) + React.js (Frontend)

# To start

1. Enter your data to .env file \
   For example:
   ![./env.png](./env.png)
2. Enter yout url to file
   [clientURL](./Shop.Client/ReactApp/src/const.ts)
3. Run
   ```
   npm install
   ```
4. Run

   ```
      nodemon
   ```
